# String Method Activity
- Research the following methods:
  * `charAt()` & `indexOf()`
  * `toUpperCase()` & `toLowerCase()`
  * `replace()`
  * `slice()`
- For each of these methods find out:
  * Syntax
  * How it is used
  * What it returns
  * When would you find this most useful?
  * Are there any "gotchas"?
  * Examples
